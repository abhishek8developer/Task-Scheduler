# Task-Scheduler
Task Assignee and Manager

## Front-End

#### React + Redux + Thunk

### Getting Started

```
npm install
npm run dev
```

Browse to `http://localhost:9000`.

### Development

```
npm run dev
```

This will start a webpack dev server with hot reloading at
`http://localhost:9000`.

### Production

```
npm run build
```

This will build and uglify files to `dist/` ready to push to production.
